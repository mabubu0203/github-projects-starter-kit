### 💡 本リポジトリ開発の背景

GitHub Projects は強力なプロジェクト管理ツールですが、初期セットアップには多くの手動操作が必要です。Field・Status・View の設定、Label の整備、Issue/PR の紐付けなど、プロジェクトを立ち上げるたびに同じ作業を繰り返すことになります。

この課題を解決するために、GitHub Projects Ops Kit を開発しました。JSON 定義ファイルと GitHub Actions を組み合わせることで、誰でも同じ構成を何度でも再現できる仕組みを提供しています。
